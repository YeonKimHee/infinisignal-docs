## [1.2] - 2026-03-14

### Added
- **탭 이름 변경 + 텔레그램 메시지 HTML 포맷 적용**
- **자동매매 탭에 활성 전략 표시 + 시뮬레이션 테스트 UI 추가**
- **자동매매 시뮬레이션 테스트 엔드포인트 추가**
- **자동매매 시스템 구현 — KIS 모의투자 주문 API + 실행 엔진 + 프론트엔드**
- **분할매수 전략 동적화 — 직전 매수가 기반 트리거 + UX 개선**
- **OG 메타 태그 + 파비콘 추가 — URL 공유 시 미리보기 카드 지원**
- **UI 대개편 — 사이드바 + 포폴/전략 통합 + 단일종목 + 코드 숨기기**
- **가격 알림 유저 분리 + 텔레그램 DM 발송**
- **뉴스 파이프라인 확장 — 종목별 요약/감성트렌드/주가연동/이슈알림**
- **뉴스 목록 기본 중요도 필터 (importance >= 0.3)**
- **뉴스 페이지 정제 상태 60초 자동 폴링**
- **뉴스 임베딩 + 클러스터링 파이프라인**
- **DART 공시 수집 파이프라인 + 배포 스크립트**
- **프론트엔드 뉴스 분석 페이지 추가**
- **뉴스 정제 파이프라인 (Gemini 2.5 Flash)**
- **시간외(AFTER_HOURS) REST 폴링 + CLOSED phase 폴링 중단**
- **AI 뉴스 파이프라인 Phase 1 (스키마 + RSS 수집)**
- **KRX 시간대별 가격 처리 (Session-Aware Pricing)**
- **Admin Status 엔드포인트 추가 (GET /api/admin/status)**
- **매도 완료 종목 재시작 기능 + 도움말 툴팁**
- **매도 체결 + 손익 계산 + 전략 알림 시스템**
- **텔레그램 봇 서비스 Docker Compose에 추가**
- **프론트엔드 빌드 통합 + HTTPS/SSL 배포 구성**

### Fixed
- 8자리 모의투자 계좌번호 지원 — 상품코드 01 자동 추가
- 잔고 조회 시 기존 KIS provider 사용하도록 수정
- Gemini maxOutputTokens 2000 → 8192로 증가
- 임베딩 모델을 gemini-embedding-001로 변경 + embedContent API 사용
- 임베딩 API를 v1 엔드포인트로 변경
- 배포 스크립트 헬스체크를 컨테이너 내부에서 실행
- RSS 한국어 인코딩 깨짐 해결 (EUC-KR/CP949 지원)
- 운영서버에서도 news DB 테이블 자동 생성
- 뉴스 수집 루프 크래시 방지 + 죽은 RSS 소스 비활성화
- RSS 피드 URL 업데이트 + SSL 검증 비활성화
- 시간외 REST 폴링 30초 → 5분 (10분 단일가 매매에 맞춤)
- 시간 표시를 UTC → KST로 수정
- CLOSED에서도 KIS API 호출하여 오늘 종가 반환
- KIS 토큰 만료 자동 재발급 + 장 마감 시 불필요한 API 호출 제거
- 초대코드 페이지 무한 리다이렉트 방지
- 환경설정에서 미사용 전략 설정 섹션 제거
- 환경설정에서 모니터링 가이드, API 기본 URL 제거
- 온보딩 가이드 모달을 로그인 후 메인 페이지에서만 표시
- /auth/callback을 SPA로 라우팅 (API가 아닌 프론트엔드)
- OAuth 세션 쿠키 SameSite=none 설정
- itsdangerous 의존성 추가 (SessionMiddleware 필요)
- 프로덕션 빌드에서 localhost fallback 제거
- 신규 DB에서 테이블 생성 후 alembic stamp 처리
- Docker 빌드 시 vue-tsc 타입체크 스킵 (vite build만 실행)
